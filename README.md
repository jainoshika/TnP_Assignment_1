# Array Problems

This repository contains solutions to fundamental and intermediate-level array problems. The goal of this collection is to build strong problem-solving skills, understand common array patterns, and prepare for technical interviews.

## Topics Covered

* Basic Array Operations
* Two Pointer Technique
* Hashing (unordered_set / unordered_map)
* Prefix Sum
* Sliding Window
* Binary Search
* Sorting-Based Approaches
* Greedy Algorithms
* In-place Array Manipulation
* Boyer-Moore Voting Algorithm
* Dutch National Flag Algorithm
